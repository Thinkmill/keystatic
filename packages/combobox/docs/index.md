---
title: Combobox
description:
  A combobox combines a text input with a listbox, and allows users to filter a
  list of options.
category: Overlays
---

## Example

```jsx {% live=true %}
<Combobox label="Combobox">
  <Item key="one">Item one</Item>
  <Item key="two">Item two</Item>
  <Item key="three">Item three</Item>
</Combobox>
```
